# ES6

##### 1. 求幂运算符(**)

```javascript
let res = 5 ** 2;//5是基数，2是幂 5的2次方 = 25
//相当于以前的
Math.pow(5,2);//25
```

##### 2. Array.prototype.includes()

```javascript
let val = [1,2,3];
console.log(val.includes(1));//true
console.log(val.includes(0));//false
```
